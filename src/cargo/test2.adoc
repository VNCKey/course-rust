= Rust + AsciiDoc Ejemplo
:source-highlighter: rouge
:icons: font
:toc: left

Pequeño proyecto que une:

* Código en Rust con doc-tests.
* Tabla de comparación (AsciiDoc).
* Generación de PDF vía GitHub Actions.

== Tabla avanzada: tamaños de escalares

[cols="1,1,1,3", options="header,autowidth", stripes=even]
|===
| Tipo
| Rust
| Bytes
| Nota

| Entero con signo
| `i32`
| 4
| El default de enteros literales

| Natural
| `u32`
| 4
| Sin signo, mismo tamaño

| Coma flotante
| `f32`
| 4
| Precisión simple

| Booleano
| `bool`
| 1
| Solo 0 o 1

| Carácter Unicode
| `char`
| 4
| Codificación UTF-32
|===

== Ejecutar

[source,bash]
----
cargo run --bin demo
----

== Documentación

[source,bash]
----
cargo doc --open
----

== PDF automático
El PDF se genera y publica en GitHub Pages al hacer push a `main`.
